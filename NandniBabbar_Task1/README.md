# Task 1 - Voice Assistant

## Description

This is a simple Voice Assistant developed in Python as part of the Oasis Infobyte Internship.

## Features

- Listens to voice commands
- Opens Google
- Opens YouTube
- Exits when the user says "goodbye"

## Technologies Used

- Python 3
- SpeechRecognition
- pyttsx3
- webbrowser
